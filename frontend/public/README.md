# Fichiers statiques

## Favicon

Placez vos fichiers de favicon ici :

- `favicon.ico` - Favicon principal (16x16, 32x32, 48x48 pixels)
- `apple-touch-icon.png` - Icône pour iOS (180x180 pixels)

## Formats supportés

- ICO (favicon.ico)
- PNG (apple-touch-icon.png)

## Génération

Vous pouvez générer un favicon à partir de votre logo en utilisant des outils en ligne comme :
- https://favicon.io/
- https://realfavicongenerator.net/

## Emplacement

Les fichiers doivent être placés dans `frontend/public/` :
- `frontend/public/favicon.ico`
- `frontend/public/apple-touch-icon.png`
