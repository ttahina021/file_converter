# Logo convifree

Placez votre fichier logo ici.

## Instructions

1. Placez votre fichier logo dans ce dossier (`frontend/public/images/`)
2. Nommez-le `logo.png` (ou modifiez le chemin dans les composants si vous utilisez un autre nom)
3. Formats supportés : PNG, JPG, SVG, WebP

## Taille recommandée

- Pour la navigation : 40x40 pixels minimum
- Pour la page d'accueil : 120x120 pixels minimum
- Format : PNG avec fond transparent (recommandé)

## Fichiers à modifier si vous changez le nom

- `frontend/app/components/Navigation.tsx` (ligne avec `/images/logo.png`)
- `frontend/app/page.tsx` (ligne avec `/images/logo.png`)
