# Backend - Convertisseur JSON vers Excel

API .NET pour convertir des fichiers JSON en Excel.

## Prérequis

- .NET 8.0 SDK ou supérieur

## Installation

```bash
dotnet restore
```

## Exécution

```bash
dotnet run
```

L'API sera accessible sur [http://localhost:5000](http://localhost:5000)

La documentation Swagger sera disponible sur [http://localhost:5000/swagger](http://localhost:5000/swagger)

## Endpoints

- `POST /api/convert/json-to-excel` - Convertit un fichier JSON en Excel
